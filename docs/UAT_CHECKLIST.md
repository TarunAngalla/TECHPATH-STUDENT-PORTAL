# Client UAT Checklist

## Admin
- Submit and review an enquiry; schedule consultation; approve and reject separate test records.
- Send/resend a secure invitation and verify older links are invalid.
- Activate an approved NDA template and review signed agreement evidence.
- Assign and reassign a recruiter; verify capacity and history.
- Launch marketing, create an application, interview, and assessment.
- Export each report type with and without a date range.

## Recruiter
- See only assigned candidates.
- Update marketing, applications, interviews, assessments, trainings, and messages for assigned candidates.
- Confirm an unassigned candidate URL is denied.

## Candidate
- Complete secure account setup and NDA signing.
- View recruiter, journey, applications, interviews, assessments, trainings, announcements, resources, and settings.
- Confirm official records are view-only and another candidate ID cannot be accessed.

## Cross-cutting
- Mobile and desktop navigation.
- Expired session, logout, suspended account, invalid invite, and provider failure messages.
- Email links point to the correct domain.
- `/api/health` and `/api/health/ready` monitored in staging.
